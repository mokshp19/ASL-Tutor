# ASL Tutor

ASL Tutor is a mobile learning app that helps users practice **American Sign Language (ASL)** through structured lessons and quizzes, with **login** and **progress tracking** so users can continue across sessions.

## Features
- ASL lessons organized into simple modules
- Quiz mode to practice and check understanding
- User authentication (sign-in/sign-up)
- Progress tracking (e.g., completion history, streaks/XP if enabled)
- Clean, mobile-friendly UI built in Flutter

## Tech Stack
- **Flutter (Dart)** – mobile UI and app logic
- **Firebase Authentication** – user sign-in
- **Cloud Firestore** – storing user progress and app data
- **GitHub** – team collaboration and version control

## Getting Started

### Prerequisites
- Install **Flutter SDK**
- Install **Android Studio** (or Xcode for iOS)
- A Firebase project (instructions below)

You can run this project either by cloning the repo or by downloading the ZIP.

### Option 1: Download ZIP (no Git required)
1. Click the green **Code** button on this GitHub page.
2. Select **Download ZIP**.
3. Unzip the folder on your computer.
4. Open a terminal in the unzipped project folder.

### Option 2: Clone the repository
```bash
git clone https://github.com/adriandbf/ASL-Tutor.git
cd ASL-Tutor
```
## Install dependencies
```bash
flutter pub get
flutter run
```
